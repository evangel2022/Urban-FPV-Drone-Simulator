# 🚁 城市 FPV 无人机模拟器

一款沉浸式的 **第一人称视角 (FPV) 无人机飞行模拟器**，完全基于 **Three.js** 构建，所有代码整合在单个 HTML 文件中。在程序化生成的城市景观中自由飞行，配备实时 HUD、碰撞检测和多种视角模式。

## ✨ 功能特性

- **程序化城市** – 随机生成的建筑、树木、路灯和道路网格
- **多种视角模式**
  - FPV（第一人称）– 驾驶舱视角
  - 追逐 – 第三人称跟随视角
  - 电影轨道 – 自由环绕视角
- **实时 HUD** – 速度、高度、电量、飞行时间、距离、碰撞计数
- **碰撞检测** – 与建筑物和地面发生物理碰撞响应
- **小地图** – 实时雷达显示无人机位置和周围环境
- **动态云层** – 飘浮的体积云效果
- **指针锁定控制** – 精准的鼠标旋转控制
- **视觉效果** – 动态阴影、建筑窗户灯光、地平线指示器

## 🎮 操作指南

| 按键 | 操作 |
|:---:|------|
| **W** | 向前飞行 |
| **S** | 向后飞行 |
| **A** | 向右平移 |
| **D** | 向左平移 |
| **Space** | 上升 |
| **Shift** | 下降 |
| **Q** | 向左旋转 |
| **E** | 向右旋转 |
| **V** | 切换视角模式 |
| **鼠标** | 环顾四周（FPV / 追逐模式） |
| **点击** | 锁定指针（FPV / 追逐模式） |

### 视角模式

- **FPV（模式 0）** – 无人机第一人称视角
- **追逐（模式 1）** – 第三人称跟随视角（*默认*）
- **电影轨道（模式 2）** – 鼠标控制的自由环绕视角

## 🚀 快速开始

无需任何构建工具！只需在浏览器中打开文件即可：

1. **克隆或下载** 本仓库
2. 在浏览器中打开 `index.html`（推荐 Chrome、Edge 或 Firefox）
3. 点击 **"启动飞行"** 开始体验
4. 点击画面锁定鼠标指针
5. 自由飞行！

> **注意：** 需要稳定的网络连接——Three.js 通过 CDN 加载。

## 🛠️ 技术栈

- **[Three.js](https://threejs.org/) r160** – 3D 渲染引擎（通过 CDN 加载）
- **原生 JavaScript（ES Modules）** – 无需构建工具，无框架依赖
- **CSS3** – 启动页面、HUD 叠加层和视觉效果
- **HTML5 Canvas** – 实时小地图渲染
